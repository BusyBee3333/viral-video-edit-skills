# Example Prompts

These prompts are starting points, not contracts. Adapt names, durations, visual roles, and delivery targets to the project.

## Build Three Social Variants

Use `$beat-synced-montage-builder` to create three `9:16` social edit concepts from this song analysis and footage inventory. Use the best `30-40s` high-energy window, keep the main action readable, and use crowd/laser/texture clips only as musical punctuation. Produce a short creative brief, planned source mix, and timeline strategy before rendering.

## Convert A Song Analysis Into An Edit Plan

Use `$beat-synced-montage-builder` to turn this `*_cut_markers_30fps.csv`, `*_sections.csv`, and footage manifest into a detailed edit plan. Identify the target window, marker density, source pools, accent slots, and final QA outputs. Do not render until the plan has a clear source taxonomy and timing strategy.

## Revise A Weak Cut

Use `$beat-synced-montage-builder` to diagnose why this beat-synced montage feels weak. Compare the timeline CSV and contact sheet against the song section energy. Recommend specific changes to source mix, accent placement, crop/layout, speed choices, and priority-hit footage.

## Replace Placeholder Accents

Use `$beat-synced-montage-builder` to replace generated laser or abstract accents with real footage. Keep accents short, full-bleed, and tied to snares, fills, drops, or phrase turns. Preserve the main edit's timing unless a better musical placement is obvious.

## Produce A Flexible Config

Use `$beat-synced-montage-builder` to create a config-driven render plan from this footage inventory. Keep the config easy to modify: source kinds, weights, accent slots, target window, format, foreground layout, and delivery outputs should be editable without rewriting the whole builder.
